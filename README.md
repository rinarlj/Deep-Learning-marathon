# Deep Learning marathon

This repository contains a collection of deep learning experiments covering **LLM evaluation, preference tuning, parallel training, and computer vision tasks**.  

## Contents  

### **1. Arm UNICEF Disaster Vulnerability Challenge**  
- Classify **roof types** in aerial images to improve disaster response in Malawi.  

### **2. LLM Evaluation**  
- Compare **GPT-2, Flan-T5-Large, LLaMA-3.1-8B, Mistral-7B, and Gemma-2-2B** on the **MMLU benchmark**.  

### **3. Preference Tuning**  
- Detoxify **EleutherAI/pythia-410m-deduped** using **Proximal Policy Optimization (PPO)**.  

### **4. Parallel Training**  
- Train **GPT-2** on **OpenWebText** using **PyTorch 2.5 FSDP** (no Hugging Face).
